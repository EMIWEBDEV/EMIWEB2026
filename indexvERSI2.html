🔐 Trust Chain Digital Signature

Visualisasi rantai kepercayaan sertifikat digital menggunakan Highcharts Network Graph

ROOT CA

Sertifikat paling atas yang menjadi sumber kepercayaan utama. Semua sertifikat lain divalidasi berdasarkan ROOT CA.

INTERMEDIATE CA

Sertifikat perantara yang menandatangani sertifikat user dan TSA. Biasanya digunakan untuk keamanan dan segmentasi trust.

SERTIFIKAT USER

Sertifikat milik pengguna untuk melakukan tanda tangan digital PDF, XML, maupun dokumen elektronik lainnya.

SERTIFIKAT TSA

Sertifikat Time Stamp Authority untuk memberikan timestamp resmi dan menjaga validitas jangka panjang (LTV).

Status Verifikasi Trust Chain

✔ ROOT CA Valid
Sertifikat ROOT CA ditemukan dan dipercaya sistem operasi.
✔ Intermediate CA Valid
Intermediate berhasil diverifikasi oleh ROOT CA.
✔ User Certificate Valid
Sertifikat user masih aktif dan belum revoked.
⚠ OCSP Response Delayed
OCSP responder lambat merespon, fallback ke CRL digunakan.
✔ TSA Timestamp Valid
Timestamp berhasil diverifikasi menggunakan sertifikat TSA.
